Privacy Policy

Data Access

The Screenplay Writer Add-on accesses the Google Document you are working with (the google doc you have open in your browser, this does not include any google document which is not open or does not have the add-on activated), it reads the content on the document, modifies the content on the document and adds new content to the document.

The Screenplay Writer Add-on also accesses and uses, the information provided by user inside a form in the sidebar ('Script Pad') offered by the Add-on.

Additionally, the Add-on accesses the user's email address to verify subscription status and track trial registration periods. This access is strictly for subscription management purposes and does not extend to any other use.

Once the google document is closed in the browser, the Add-on cannot read, modify or add any content to the google doc.

Data Usage

The Screenplay Writer Add-on uses the content of the google document you are working with, to apply the different formatting changes to the document to meet standard Screenplay Format.

The Add-on uses the data that is given by user inside the form in the sidebar provided by the Add-on, to add and modify the content of the Google Document the user is working with.

The data is used only for modifying and formatting the current working google document, and not brought into use for any other purposes at all.

Additionally, the Add-on accesses the user's email address to verify subscription status and track trial registration periods. This access is strictly for subscription management purposes and does not extend to any other use.

These details are used solely for subscription management purposes and are not shared or used for any other purpose.

Data Storage

The Screenplay Writer Add-on, does not store any data, we do not store any personal or document-related information in any capacity. Your work remains entirely private and not stored by Screenplay Writer or any third parties.

The Add-on does not share or sell user data, including email addresses, to any third parties. Furthermore, it does not transmit document content or user data to any external service except for the explicit purpose of subscription verification.

Data Protection

The Screenplay Writer Add-on employs secure methods to manage and protect stored user data. User email addresses are stored in a secure database accessible only to authorized personnel and systems.

The Add-on does not share or sell user data, including email addresses, to any third parties. Furthermore, it does not transmit document content or user data to any external service except for the explicit purpose of subscription verification.

User data, including email addresses and document content, is protected under stringent security measures to prevent unauthorized access or misuse.

The Screenplay Writer Add-on does not store the data it accesses in the current working google document. Neither does it pass on the accessed information to any 3rd party services.

Data Retention/Deletion

The Screenplay Writer Add-on retains user email addresses for the duration of the user's active subscription or trial period. Once the subscription is canceled or the trial period ends, the user has the option to request the deletion of their data.

Users can delete their data by contacting us at betashorts1998@gmail.com.

Data deletion requests are processed promptly, and all associated data is permanently removed from our systems.

The Screenplay Writer Add-on does not store the content in Google Documents. So, there is no data retention or data deletion required for those cases

Contact Details

If you have any questions or concerns regarding this privacy policy, please contact us at: betashorts1998@gmail.com

Cookies and Tracking

The Screenplay Writer Add-on does not use any cookies, log files, or tracking mechanisms. We do not monitor or track any user activity while using the Add-on.

Opting Out

Users can discontinue using the Add-on at any time by removing it from their Google Docs account. Additionally, users may request to delete their stored data by using the "Delete User Data" option in the sidebar or contacting support at the email address provided above.

Third-Party Services

This Add-on does not integrate with any third-party services that collect user data. Any interaction with external services (e.g., through Google Docs) is governed by the respective third-party’s privacy policy. Screenplay Writer disclaims any responsibility for the privacy practices of such services.

Google Docs Data

Google Docs does not use content created within its environment for advertising purposes. This Add-on operates entirely within Google Docs, and all document content is governed by Google’s privacy policies. The Screenplay Writer Add-on does not interact with this data beyond assisting with formatting tasks.
